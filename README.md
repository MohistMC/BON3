BON3
====

A rewrite for Immibis's bearded-octo-nemesis for ForgeGradle by tterrag, with features by LexManos.

### Based on https://github.com/GTNewHorizons/BON2

## Our modifications
- JDK-17
- gradle-8.0.2
- Numerous Java language logic and level modifications
